# Roadmap MTIT-OS

## Fase 1: Base operativa

Crear estructura, roles, estándares, políticas mínimas, playbooks iniciales y registro de proyectos.

## Fase 2: Automatización con GitHub Issues y PR

Convertir estándares y playbooks en flujos trazables con issues, checklists, PRs, revisiones y estados.

## Fase 3: Plantillas reutilizables para nuevos proyectos

Crear plantillas para nuevos repositorios, proyectos, clientes, QA, seguridad, documentación y operación.
